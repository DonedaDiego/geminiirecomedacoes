Bandas de Volatilidade Híbridas

Este indicador foi desenvolvido com base em modelos estatísticos e técnicas modernas de machine learning, com o objetivo de identificar zonas potenciais de suporte e resistência utilizando a volatilidade como referência.

Inteligência Artificial Aplicada ao Trading

Nosso modelo captura a volatilidade passada usando estimativa ponderada com GARCH, depois roda machine learning XGBoost com outras 15 variáveis para estimar suportes e resistências com precisão matemática.

Modelo GARCH

Generalised Autoregressive Conditional Heteroskedasticity - padrão mundial para volatilidade financeira.

Características:
  • • Volatilidade Condicional: Adapta-se ao histórico
  • • Clustering: Períodos de alta/baixa volatilidade
  • • Mean Reversion: Tendência de retorno à média
  • • Robustez Estatística: Validado academicamente

XGBoost ML

Extreme Gradient Boosting - algoritmo de machine learning mais avançado para previsões.

Vantagens:
  • • 15 Variáveis: Análise multidimensional
  • • Gradient Boosting: Aprendizado iterativo
  • • Feature Selection: Identifica padrões ocultos
  • • Alta Precisão: Minimiza erro de previsão
